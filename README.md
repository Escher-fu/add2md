# add2md

The goal of add2md is to help R users to type markdown in `.Rmd`.

## Installation

You can install the released version of add2md from
[Github](https://github.com/JiaxiangBU/add2md) with:

``` r
devtools::install_github('JiaxiangBU/add2md')
```

``` r
library(add2md)
```
